# BTD6 Mod Manager - Quick APK Generation Guide

## 🚀 Fastest Way to Get Your APK

Since you don't have a PC with Android Studio, here are the easiest options:

### Option 1: Use Expo EAS Build (Recommended - 5 minutes)

**No setup needed! Cloud-based building.**

1. **Sign up for free accounts:**
   - Expo: https://expo.dev (free tier)
   - GitHub: https://github.com (free)

2. **Push your code to GitHub:**
   - Create a new repository
   - Push this project to GitHub

3. **Install EAS CLI:**
   ```bash
   npm install -g eas-cli
   ```

4. **Build your APK:**
   ```bash
   eas login
   eas build --platform android
   ```

5. **Download your APK:**
   - Check your email for download link
   - Or visit https://expo.dev/builds

### Option 2: Use Online APK Builders

Several free online services can build APKs from web apps:

- **Apache Cordova Online**: https://build.phonegap.com
- **Capacitor Cloud Build**: https://capacitor.ionicframework.com/docs/guides/ci-cd
- **PWA to APK**: https://www.pwabuilder.com

### Option 3: Ask Someone with Android Studio

If you know someone with Android Studio and Android SDK:
1. Give them the `APK_BUILD_GUIDE.md` file
2. They can build it locally in 10 minutes

## 🎮 Using the App

1. **Install the APK on your Android phone**
   - Download the APK file
   - Enable "Unknown Sources" in Settings
   - Tap the APK to install

2. **Launch the app**
   - Find "BTD6 Mod Manager" in your app drawer
   - Tap to open

3. **Install mods**
   - Browse available mods
   - Click "Install Mod" button
   - App checks for BTD6 Mod Helper
   - Mods are installed automatically

4. **Launch BTD6**
   - Open Bloons TD 6
   - Mods will be loaded automatically

## 🔧 Troubleshooting

### "BTD6 Mod Helper Not Detected"
- Make sure BTD6 Mod Helper is installed
- Restart the app
- Check that the mods directory exists

### "Installation Failed"
- Check your internet connection
- Ensure you have enough storage space
- Try downloading a different mod

### "App Won't Open"
- Clear app cache in Settings
- Uninstall and reinstall the APK
- Check that you have Android 6.0+
