---

## 📱 Step 1: Open Terminal/Command Prompt

**On your phone or computer:**
- Open a terminal (Mac/Linux) or Command Prompt (Windows)
- Make sure you have Node.js installed: https://nodejs.org/

---

## 🔧 Step 2: Install EAS CLI

Copy and paste this command:

```bash
npm install -g eas-cli
```

Wait for it to finish.

---

## 🔑 Step 3: Login to Expo

Copy and paste this command:

```bash
eas login
```

When prompted:
- **Username:** `milkvryt`
- **Password:** (your Expo password)

---

## 🚀 Step 4: Build Your APK

Copy and paste this command:

```bash
eas build --platform android
```

**What happens next:**
1. The build starts in the cloud
2. You'll see a link like: `https://expo.dev/builds/...`
3. Click it to watch the build progress
4. Wait 5-10 minutes for the build to complete

---

## 📥 Step 5: Download Your APK

Once the build finishes:

1. Go to: https://expo.dev/builds
2. Find your build in the list
3. Click the download button
4. The APK file will download

---

## 📲 Step 6: Install on Your Phone

1. **Transfer the APK to your phone** (via email, cloud storage, etc.)
2. **Open the APK file on your phone**
3. **Enable "Unknown Sources"** if prompted:
   - Settings → Security → Unknown Sources → Enable
4. **Tap "Install"**
5. **Done!** The app is now installed

---

## 🎮 Step 7: Use the App

1. Open your app drawer
2. Find **"BTD6 Mod Manager"**
3. Tap to launch
4. Browse and install mods!

## 🆘 Troubleshooting

### "Command not found: eas"
- Make sure Node.js is installed
- Restart your terminal after installing EAS CLI

### "Build failed"
- Check your internet connection
- Try running the command again
- Check the build logs at https://expo.dev/builds

### "APK won't install"
- Enable "Unknown Sources" in Settings
- Make sure you have enough storage space
- Try uninstalling any previous version first

### "App crashes on launch"
- Clear the app cache in Settings
- Uninstall and reinstall
- Check that you have Android 6.0 or higher
